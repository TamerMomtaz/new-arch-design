The Four Validation Gates

Quality checkpoints with human-in-the-loop where it matters most

Gate 1

Data Validation

Automatic

Ensures SME data is complete and valid before any agent processes it. Checks questionnaire completeness, data types, and critical fields.

Gate 2

Confidence Check

Auto + Escalation

Each agent's output is scored for confidence. If below 0.7 threshold, routes to human advisor for review before proceeding.

Gate 3

Conflict Resolution

HITL Conditional

Detects contradictions between agent recommendations. The Drucker attempts automatic resolution; unresolved conflicts go to human review.

Gate 4

Quality Assurance

HITL for High-Stakes

The Deming validates final report. Human approval required for: first-time SMEs, donor-facing reports, critical scores, and major recommendations.
